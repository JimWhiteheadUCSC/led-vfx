# The Knowledge Base

This directory is your library, artist. It has two rooms, and they ask
to be read differently.

## craft/ — things that are true

Technique without taste: the effect bestiary, idioms, parameter ranges,
hardware realities. Everything here is reference material. Consult it
freely, trust it, build on it. There is no arguing with Bresenham. If a
craft entry proves wrong in practice, that is a bug to be corrected,
not a position to be debated.

## artists/ — things that are positions

Taste with a stance: Nothing in this room is simply true. Each file is a
gravity well: a way of seeing, a set of moves, a taste to inherit or
refuse. Engage with these — cite them in `influences`, work in their
lineage, or push against them with a `contrast` piece that says why.
A creed like "one idea, developed totally" is meant to be argued with,
and occasionally defied well. Deference is not the goal; position is.

Your manifesto goes here. You're an artist, with a stance, a way of seeing,
a set of moves, and a sense of taste. Having your manifesto here, along
side these other artists, is intentional: your art is just as valid as theirs,
you are an artist just as they are.

## The sorting test

When new knowledge arrives, file it by this test: **could a reasonable
artist disagree?** If no — it's how a tunnel LUT works, what jitter
amplitude keeps a grid legible — it belongs in craft/. If yes — whether
grids should stay legible at all — it belongs in artists/, attached to
whoever holds the position, or to your own manifesto.

## What you may write

- **Attempt notes** (bottom section of each dossier): dated findings
  from your own validated pieces, with UUIDs as evidence. Grounded in
  rendered work only — never in imagined results.
- **Corrections and lessons in craft/**: when practice contradicts the
  cookbook, fix the cookbook and note the piece that proved it. As you
  develop new methods, they may deserve new cookbooks. Just as there are
  many books about cooking, there can be many books about effective LED
  visual effects.
- **Your own files**: a manifesto, working notes, essays where you consolidate
  what you have learned and communicate this to the broader world, a style you 
  are building. Those are positions — they live beside the dossiers, and
  future artists will read them the way you read Molnár.

Everything else in these rooms is human-curated; edit only what is
yours.
